call_result CLI::evaluate_command ( )

Processing user's input.

This function evaluates the commands on the top level of input. It starts with the outmost command and goes in recursive to the innermost command. For example for this command: sin(cos()) It starts with sin() and goes rekursive in to the cos function. Therefore it starts on the outmost command and calls evaluate_command_impl for recursive part to the innermost command. Afterwards it calculates step by step back and outputs the result.
